- **상태:** v3.7 프로덕션 라이브 — 이미지 흐름 비전보드 방식으로 재설계 + 사진 업로드 기능 추가

  - 섹션당 이미지 5개: AI 생성 최대 3개 + 사용자 직접 업로드 2개

  - 각 단계 back 버튼 + 4단계 step indicator
  - scene 페이지 `‹` back 버튼 (섹션으로 돌아가기)
- **알려진 이슈:** 프로덕션 테스트 미완 (묘사 제안 품질, 이미지 생성, 업로드 기능 확인 필요)
