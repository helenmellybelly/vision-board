## 변경된 것

### Step 3 — 비전보드 소개 버튼
- **v0.1:** [응, 들어봤어] / [아니, 처음이야] 두 분기 버튼
- **구현본:** 단일 버튼 "오… 그래서 어떻게 하는 건데?" — 분기 불필요, 다음 단계로만 이동
- **이유:** 어느 쪽이든 같은 화면으로 가므로 버튼 2개가 불필요한 선택 마찰 생성

### Step 4 — 상태 공감 선택지
- **v0.1:** 선택지로 받되 분기용 아님 (공감용), 방식 미결
- **구현본:** [잘 안 그려져] / [더 생생히 그리고 싶어] 버튼 + "맞아, 딱 내 얘기야" 텍스트 skip
- **이유:** 선택을 받되 어느 쪽이든 같은 Step 5로 이동. skip은 읽고 넘길 수 있게.

### 비전보드 예시 이미지 (Step 3)
- **v0.1:** 예시 보드 1장 (추후 제작 예정) — placeholder 처리
- **구현본:** placeholder 유지 ("예시 비전보드" 텍스트 박스)
- **미결:** 실제 예시 이미지 1장 필요 (분위기: 미니멀 or 따뜻함 결정 후 제작)

---

## 미결 사항 (다음 버전에서 해결)

1. **예시 보드 이미지** — 어떤 분위기로 제작할지 결정 필요
2. **lumi 아바타/아이콘** — 텍스트 말풍선만 있음, 시각적 캐릭터 여부 결정
3. **재방문 시 온보딩 skip** — 현재 랜딩에서 "계속하기" 버튼으로 처리, 별도 인사말 없음
