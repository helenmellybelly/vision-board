---

## 구조 안내

각 섹션은 4개 PHASE로 진행됩니다.

```
PHASE 1 — 슬롯 ① → ② → ③ → ⑤ (질문 답하기)
PHASE 2 — ①②③⑤ 답 한눈에 모아보기 + 수정
PHASE 3 — 슬롯 ④ (②를 보며 장면 묘사)
PHASE 4 — 슬롯 ⑥ (이미지 3장 선택)
```

각 슬롯 구성:
- **메인 질문** — 화면에 크게 표시
- **Placeholder** — 입력창 안내 문구 (회색 글씨)
- **예시** — 질문 아래 작은 글씨 (클릭하면 펼쳐짐)
- **도움이 필요해요** — 버튼 탭 시 서브질문 펼침 (막힌 경우 사용, AI 없이 고정 질문)

**부제:** 어떤 사람으로, 어떤 감각으로 살고 싶은지  
**색상:** 자주 (#8B5CF6)

### 슬롯 ① 지금 (PHASE 1)

**메인 질문**
> 지금 나는 어떤 사람으로 살고 있어? 한 문장으로.

**Placeholder**
> 좋은 답 나쁜 답 없어. 솔직한 게 제일 좋아.

**예시**
> 그냥 하루하루 버티는 사람 / 열심히 사는데 뭔가 빠진 느낌 / 남들 눈에는 잘 사는데 속으론 모르겠는 사람

**도움이 필요해요 (서브질문)**
1. 요즘 아침에 일어날 때 가장 먼저 드는 감정이 뭐야?
2. 지금 내 삶에서 가장 많은 에너지를 쓰는 게 뭐야?
3. 주변 사람들이 나를 어떻게 보는 것 같아? 그게 맞는 것 같아?

**예시**

3. 지금 제일 갖고 싶은 감각이 뭐야? (여유, 자신감, 편안함...)

**도움이 필요해요 (서브질문)**
