> v0.5→v0.6 수정: 페이스 방식(강제 단일→자율 페이스), 섹션 질문 템플릿 추가.
> **2026-05-27 업데이트:** ① 섹션 내부 플로우 순서 확정 (PDF 실제 흐름 기반), ② 박지현 실제 답변 예시 추가, ③ 장면으로 슬롯이 답 리뷰 이후에 위치함을 명시.
> 기반: 박지현 님이 남편(노호성)과 종이로 진행한 **비전보드 원본 PDF**.

## 0. 한 줄 정의 / 핵심 가치

> **막연한 감정을 → 진짜 원하는 삶의 구체적 장면으로 → 매일 보고 싶은 한 장의 이미지로 바꿔준다. 번아웃 없이, 본인 페이스로, '진짜 내 답'이 나올 때까지.**

차별점:
- 파는 건 예쁜 보드가 아니라 **'질문'** (핀터레스트도 보드는 만든다. 우리 IP는 "나도 모르던 걸 말하게 하는" 질문 흐름.)
- 목표 설정이 아니라 **정체성 우선** ("나는 어떤 사람으로, 어떤 *기분*으로 살고 싶은가")
- 최소 단위는 목표가 아니라 **감정/상태** (방향 키워드가 죄다 *여유로운·단단한* 같은 느낌 단어인 이유)

구조 역할 분담: **엔진**(1주차 질문 흐름) / **차체**(비전보드 6섹션) / **변속기**(본인이 정하는 페이스·저장/재개) / **결과물**(이미지 보드 + 통합 1장)

### 0-1. 플랫폼 (빌드 전제)
- **웹으로 만든다.** 별도 앱 아님.
- **모바일 웹 최적화 필수.** 사용자가 폰 브라우저로 여는 걸 기본으로 가정 → 반응형, 터치 친화적 UI, 작은 화면에서 질문·입력·이미지가 편하게 보이게.
- 저장/재개·체크인 등도 모바일 웹에서 매끄럽게 동작해야 함.

## 1. 설계 대원칙

1. **강요하지 말고 본인이 발견하게.** AI는 새 내용을 심지 않는다. 항상 본인 재료로만 거든다.
2. **가짜 원함을 '걸러내려' 하지 않는다.** 몰입을 최대로 → 진위는 깊이가 알아서 드러낸다.
3. **언어는 일상적이고 받아들이기 쉽게.** 진단·심사 말투는 몰입을 깨는 1순위.
4. **언어 먼저, 이미지 나중.**
5. **펜은 항상 사용자가.** AI는 막힌 순간에만 손을 잡아준다.

### 1-1. 페이스
- **강제 휴지기 없음.** 언제든 저장하고 나가기 / 이어서 하기.
- 중간 체크인(섹션 사이): *"지금 이런 비전을 그리고 계시군요. 속도 괜찮아요? 저장하고 다음에 와도 돼요."*
- 속도는 사용자가 정한다. 우리는 "천천히 해도 된다"는 허락만 준다.

## 2. 전체 플로우

```
인트로 ("6개로 쪼갰어요. 막히면 같이 찾아드릴게요. 속도는 알아서 — 저장하고 와도 돼요.")
   ↓
[섹션 1~6] 각 섹션 아래 순서로 진행:

   PHASE 1 — 질문 답하기
   ① 지금 (현재 상태·감정)
   ② 방향 키워드 (3년 뒤 느낌 단어)
   ③ 원해! (버킷리스트·되고 싶은 모습)
   ⑤ 더 들여다보기 (깊이 질문)
   → 각 질문에서 막히면 "도움이 필요해요" → AI 서브질문 (답은 본인이)

   PHASE 2 — 내 답 한눈에 보기 · 수정
   → ①②③⑤에서 쓴 내용을 한 화면에 모아 보여줌
   → 수정 가능

   PHASE 3 — 장면 그리기
   ④ 장면으로: 위 답들을 보면서 "그 단어가 느껴지는 하루" 장면 묘사
   → 막히면 AI 장면 제안 (본인 ②③ 답 재료로만 / 선택·수정·빈칸형)

   PHASE 4 — 이미지 찾기
   ⑥ 이미지 찾기: 3장 (AI 검색어 제안)

   → [섹션 사이] 가벼운 체크인 + 저장
   ↓
18장 다 모임 → 슬라이드로 천천히 → 전체 펼침
   ↓
[마무리] (1) AI가 '말'로 패턴 비춤 → 본인이 한 문장 확정
         (2) 그 문장으로 통합 이미지 1장
   ↓
완성된 나의 비전보드
```

> **슬롯 순서 근거:** 종이 PDF 실제 작성 흐름 확인. ①②③⑤(답하기) → 답 확인 → ④(장면)이 자연스럽다. "장면 그리기"는 앞 질문들의 답을 모두 본 뒤에야 제대로 나온다.

## 3. 섹션 질문 템플릿

**6섹션:** 1 나(감정·성장·정체성) / 2 건강(몸·마음·에너지) / 3 관계·가족 / 4 일·성장 / 5 돈·생활 / 6 공간·환경

**한 섹션 = 아래 6슬롯, 진행 순서는 ①②③⑤ → [리뷰] → ④ → ⑥.**

| 슬롯 | 내용 | 진행 타이밍 |
|------|------|-------------|
| ① 지금 | 현재 상태·감정 (미래 안 묻고 현재부터 — 막막한 사람도 진입) | PHASE 1 |
| ② 방향 키워드 | 3년 뒤를 한 단어로 (느낌 단어) | PHASE 1 |
| ③ 원해! | 버킷리스트 / 되고 싶은 모습 | PHASE 1 |
| ⑤ 더 들여다보기 | 깊이 질문. 1주차 출처·동기·기분 질문을 '더 빠져들게' 프레임으로 | PHASE 1 |
| ④ 장면으로 | ①②③⑤ 답 리뷰 화면 본 뒤 → 그 단어가 느껴지는 하루 장면 묘사 | PHASE 3 |
| ⑥ 이미지 찾기 | 3장 (AI 검색어 제안) | PHASE 4 |

**공통:** 모든 질문에 "도움이 필요해요" 버튼 → AI 서브질문 펼침 (답 대신 주지 않음).

**④ AI 장면제안 규칙:** 본인 ②③ 답을 재료로만. 받는 법 3가지 — ⓐ선택 ⓑ수정 ⓒ빈칸형.

## 3-1. 6섹션 실제 질문 + 예시 (원본 PDF 원문 확정 + 박지현 실제 답변)

> 각 슬롯: **질문** / *예시 또는 박지현 실제 답변 (★)* / [진행 타이밍]

### 섹션 1 · 나 — 감정·성장·정체성
- ① 지금 [P1]: **지금 나는 어떤 사람으로 살고 있어? 한 문장으로.**
  - *예: 그냥 하루하루 버티는 사람 / 열심히 사는데 뭔가 빠진 느낌*
  - ★박지현: "몸과 마음을 챙기며, 나를 표현하는 연습을 하며 경제적 자유를 준비하는 사람"
- ② 키워드 [P1]: **3년 뒤의 나를 한 단어로 표현해봐.**
  - *예: 여유로운 / 단단한 / 설레는 / 자유로운*
  - ★박지현: "단단하고 트임"
- ③ 원해! [P1]: **나를 위해 언젠가 꼭 해보고 싶은 것, 되고 싶은 모습이 있어?**
  - ★박지현: 내 성격 담긴 브랜드 / 영어로 강의 듣고 자유롭게 말하기 / 이미지컨설팅 받기
- ⑤ 더 들여다보기 [P1]: **그게 이뤄진 나는 어떤 기분으로 하루를 살고 있을 것 같아?**
  - ★박지현: "충만함·편안함으로 하루를 내 대로 잘 활동하고 싶어. 나를 평등할 줄 아는 사람."
- — **[P2] 내 답 한눈에 보기** —
- ④ 장면으로 [P3]: **방금 쓴 단어가 느껴지는 하루. 어디서, 뭘 하고 있어?**
  - *예: 카페 창가에서 혼자 책 읽는 중 / 아무 계획 없는 주말 아침*
  - ★박지현: "내 브랜드로 파생된 사람들을 한자리에 모아. 재택에서 나의 일을 시작하며, 내 메일엔 업체들만 있고, 강연 요청도 들어와. 재무적 성장도 하고 있어."
- ⑥ 이미지 [P4]: 검색 힌트 *"여유로운 아침" "혼자만의 시간" "나만의 루틴"*
  - ★박지현: 나만의 서재 / 좋은 이메일 알림 / 내 SNS·웹페이지 / 고객 후기 / 영향력 있는 모습

### 섹션 2 · 건강 — 몸·마음·에너지
- ① 지금 [P1]: **지금 내 몸 상태, 10점 만점에 몇 점? 어떤 부분 때문에 그 점수야?**
  - ★박지현: "7점 — 술 안 마심, 건강한 아침식사, 비타민·약 챙김. 3점 빠진 이유 — 운동·스트레칭 없음"
- ② 키워드 [P1]: **3년 뒤 내 몸과 나의 관계, 어떤 한 단어면 좋겠어?**
  - ★박지현: "가벼운 평안함"
- ③ 원해! [P1]: **해보고 싶은 운동, 도전해보고 싶은 것, 몸에 관해 이루고 싶은 것.**
  - ★박지현: 요가 프로필 사진 / 10KM 런닝 / 수영 / 줄넘기
- ⑤ 더 들여다보기 [P1]: **몸이 가뿐하고 에너지 넘칠 때, 지금은 못 하지만 하고 싶어지는 게 있어?**
  - ★박지현: 트래킹 / 집안 근육 바꾸기 / 집안 정비 완하기
- — **[P2] 내 답 한눈에 보기** —
- ④ 장면으로 [P3]: **방금 쓴 단어가 느껴지는 순간. 어디야? 뭘 하고 있어?**
  - ★박지현: "자고 일어났을 때 숨이 편안하게 다니는 길 느끼고 있어. 속 편한 음식을 먹고 가볍게 산책하고 있어."
- ⑥ 이미지 [P4]: *"서핑 라이프" "등산 뷰" "새벽 러닝" "활기찬 아침"*
  - ★박지현: 마라톤 결승선 / 내가 가장 마음에 드는 모습 / 자연 속 쉬는 모습 / 숲길 산책 / 좋아하는 스타일로 입은 모습

### 섹션 3 · 관계·가족 — 남편·가족·친구
- ① 지금 [P1]: **10년 뒤, 소중한 사람이 나를 떠올릴 때 어떤 사람으로 기억됐으면 좋겠어?**
  - ★박지현: "나를 진정으로 생각하고 교류해주는 사람"
- ② 키워드 [P1]: **내가 원하는 관계의 느낌, 한 단어로.**
  - ★박지현: "교류"
- ③ 원해! [P1]: **관계나 가족에 관해 꼭 이루고 싶은 것, 함께하고 싶은 것이 있어?**
  - ★박지현: 우리만의 집 → 처음처럼 하나씩 함께 / 둘 다 안 해본 경험 함께하기
- ⑤ 더 들여다보기 [P1]: **지금 가장 소중한데 더 가까워지고 싶은 사람이 있어? 그 사람이랑 어떤 시간을?**
  - ★박지현: "언니. 긍정적이고, 자랑스럽고, 편한 대화."
- — **[P2] 내 답 한눈에 보기** —
- ④ 장면으로 [P3]: **방금 쓴 단어가 느껴지는 관계의 장면 하나. 누구랑, 어디서?**
  - ★박지현: "각자의 생각에 대해서 '그렇구나!' 하고 도전하는 것을, 곁에서 들어주는 것."
- ⑥ 이미지 [P4]: *"가족 여행" "커플 여행" "따뜻한 식탁"*

### 섹션 4 · 일·성장 — 커리어·배움·의미
- ① 지금 [P1]: **지금 일에서 제일 아쉬운 게 뭐야? 딱 하나만.**
  - ★박지현: "단순반복적인 것"
- ② 키워드 [P1]: **일하는 나를 한 문장으로 써봐. 어떤 사람으로 일하고 싶어?**
  - ★박지현: "내 안에 있는 걸 활용하고, 배우며 성장하는 사람"
- ③ 원해! [P1]: **일이나 성장에 관한 버킷리스트가 있어?**
  - ★박지현: 내 브랜드 상업화 / 내 프로그램 운영 / 글로벌 회사 출장
- ⑤ 더 들여다보기 [P1]: **월요일 아침, 살짝이라도 기대되는 날 — 어떤 일을 하고 있어?**
  - ★박지현: "제안 메일에 대해 당당하고 있어."
- — **[P2] 내 답 한눈에 보기** —
- ④ 장면으로 [P3]: **방금 쓴 문장 속의 내가 하루를 마무리하는 장면. 어디야? 어떤 기분?**
  - ★박지현: "내가 배웠던 것들이 잠재되어 있다가 효율로 나타나겠다고 느끼며, 경험력을 쌓아가."
- ⑥ 이미지 [P4]: *"재택근무 라이프" "카페 작업" "내 브랜드"*
  - ★박지현: 유료 크리에이터 모습 / 고객 후기 / 맥북·업무시스템 / 일의 비전보드

### 섹션 5 · 돈·생활 — 재정·소비·여유
- ① 지금 [P1]: **지금 돈 때문에 참고 있거나 못 하고 있는 게 있어? 솔직하게.**
  - ★박지현: 요가 / 북유럽 여행 / 이미지 컨설팅 / CPCC 자격
- ② 키워드 [P1]: **돈이 충분할 때 느끼고 싶은 감정, 한 단어로.**
  - ★박지현: "안정"
- ③ 원해! [P1]: **돈 버킷리스트가 있어?**
  - ★박지현: 하와이·제주 한 달씩 / 숲view·역세권·수영장 있는 내 집 마련 / 해외 여행 경험
- ⑤ 더 들여다보기 [P1]: **돈 걱정이 딱 사라진다면, 이번 주를 어떻게 보내고 싶어?**
  - ★박지현: 집안 물건 싹 바꾸기 / 요가 / 이미지 컨설팅 / 언론 관리 받기
- — **[P2] 내 답 한눈에 보기** —
- ④ 장면으로 [P3]: **방금 쓴 단어가 느껴지는 순간. 어떤 상황이야?**
  - ★박지현: "지금 내가 당당하게 돈을 써도 불안하지 않으며, 평온하게 이것저것 하고 여행도 가는 것."
- ⑥ 이미지 [P4]: *"하와이 여행" "내 집 마련" "여유로운 주말"*

### 섹션 6 · 공간·환경 — 집·동네·일상의 배경
- ① 지금 [P1]: **집에 돌아왔을 때 제일 먼저 하고 싶은 게 뭐야? 지금 그게 가능해?**
  - ★박지현: "공간 정돈된 곳에, 옷 걸어두고, 가방 정리 → 지금도 가능. (침실·서재·안방·거실·화장방)"
- ② 키워드 [P1]: **내가 원하는 공간을 딱 한 단어로 표현하면?**
  - ★박지현: "심플"
